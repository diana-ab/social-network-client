# 🚀 Social Network Client

Welcome to **The Empire** client side 👑  
This is the frontend part of the project.

---

## 📦 Project Setup

Follow these steps to run the project locally:

### 1. Install dependencies

Open the terminal inside the project folder and run:

```bash
npm install
```

This command installs all required packages and sub-dependencies from `package.json`.

---

### 2. Start the development server

Run:

```bash
npm run dev
```

After that, Vite will start the client locally and show you the local development URL in the terminal.

---

## 🔗 Important – Server Side Required

⚠️ This client project will **not work properly without the server project running**.

You must also clone and run the server side project from:

`https://github.com/Shimon98/social-network-server-master.git`

Make sure the backend is running before testing the client.

---

## 🖥️ Recommended Run Order

1. Run the **server**
2. Run the **client**
3. Open the local Vite URL in the browser

---

## ⚠️ Troubleshooting

If something does not work:

- Make sure you ran `npm install`
- Make sure the server project is running
- Make sure both projects are running at the same time
- Check the terminal for errors
- If needed, restart the client with `npm run dev`
